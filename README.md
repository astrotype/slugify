# Транслитерация URL по правилам Яндекса

[![License](https://img.shields.io/badge/license-MIT-green)](LICENSE)

Транслитерация русских слов в латиницу по правилам Яндекса, генерация SEO-дружественных slug.

---

## Использование

Просто скопируйте файл slugify.ts в ваш проект

```ts
import slugify from './slugify'

console.log(slugify('Екатеринбург')) // yekaterinburg
```
## Особенности
- Преобразует все русские буквы (а–я, включая ё) в латиницу.
- `е → ye` в начале слова, после гласной или после ъ.
- Окончания `-ый/-ий` → `iy`.
- Мягкий знак перед `и` → `y`.
- Пробелы и спецсимволы заменяются на -, множественные - сжимаются, лишние обрезаются.
- SEO-дружественные URL для сайта или приложения.
